# Контрол с кнопками
Текстовый контрол, который позволяет настраивать и выводить разное кол-во кнопок слева и справа от самого контрола. Для кнопок должна быть возможность настроить текст и колбек функцию, которая вызывается при нажатии на кнопку.

В основном компоненте приложения нужно сделать два контрола для тестирования:

```
Контрол с 2 кнопками справа;
При нажатии на первую кнопку очищается содерживое в контроле;
При нажатии на вторую кнопку текст в контроле меняется на 'Hello world!';
```
```Контрол с 1 кнопкой справа и 1 кнопкой слева;
При нажатии на кнопку справа вызывается alert с текстом в контроле;
При нажатии на кнопку слева проверяем, что в контроле введено число и если это так, то выводим число через alert;
```
---
## `Как это сделать`
Можно было сделать одну компоненту, которая принимает список кнопок слева и кнопок справа. По этим данным отрисовать кнопки (**справа**/**слева**) передав им имя и функцию. Можно было сделать отрисовку кнопок исходя их данных `input`, но нам понадобится работа с его содержимым, поэтому, этот вариант отпадает.

## `Как это можно улучшить`
Скорее всего, смена имени кнопки и указание функции должно было указываться в самой компоненте, но можно сделать лучше. Можно менять как имя кнопки так и саму функцию сразу через поля ввода для каждой кнопки. Чтобы из строки преобразовать функцию, понадобится метод из JS `eval()`. Конечно, это очень не безопасно, так как пользователь может вшить инъекцию и навредить работе приложения, поэтому, можно будет проигнорировать ввод функции пользователем, если в массиве для кнопок передана функция.
